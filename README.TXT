Para executar o processo via linha de comando:
1. Ir para o diretório do projeto
2. no Windows: java -jar "Projeto2.jar"
   no Linux: javac Projeto2.java
3. Localizar o arquivo cep_ordenado.dat
4. inserir o CEP que deseja procurar.

Recomendado utilização via terminal para ter acesso a informações adicionais.
