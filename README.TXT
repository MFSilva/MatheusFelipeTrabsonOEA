Para executar o processo via linha de comando:
1. Ir para pasta do projeto
2. no Windows: java -jar "Projeto2.jar"
   no Linux: javac "Projeto2.java"
3. Localizar o arquivo cep_ordenado.dat

Alternativa: Executar o jar encontrado na pasta "Dist".
Este não mostra os dados adicionais como total de iterações no loop e afins.
